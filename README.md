# Homebrew repository for libkrun, krunkit, krunvm an krunai

This is a Homebrew repository for:
- [libkrun](https://github.com/containers/libkrun/) and its dependencies
- [krunkit](https://github.com/containers/kunkit/)
- [krunvm](https://github.com/containers/krunvm/)
- [krunai](https://github.com/slp/krunai/)

## Supported platforms

The formulae and bottles on this repository only support Apple Silicon-based devices.

## How do I install these formulae?

```
brew tap slp/krun
brew install krunkit
brew install krunvm
brew install krunai
```
